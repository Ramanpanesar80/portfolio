Medical Website

This project was created in react js. it is a medical website, which is designed for doctors and patients. The website lets you book an appointment with a doctor at a selected date and time. Futher, the website also lets you pay for the appointment.An API(stripe) is used for payment handling. Last but not least, the website also lets you write feedback to the doctor.
