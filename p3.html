Perfume Website

This project was created using html, css and javascript. It is a perfume website, which is designed for users to view perfumes and their details. The user can filter the perfumes by categories. The user can also search for perfumes by name. Futher, the website also lets you add perfumes to the cart. The website also has a login and signup page, a contact us page and an about us page.
