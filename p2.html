Movies Website

This project was created in html, css and javascript. It is a movies website, which is designed for users to view movies and their details. The website lets you view movies and their details. Futher, the website also lets you search for movies by name. The website also has a login and signup page, a contact us page and an about us page. The website also has a admin page, which is only accessible to the admin.
